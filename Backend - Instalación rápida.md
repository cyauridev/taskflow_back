Pasos para ejecutar el backend
Clonar el repositorio:
git clone https://github.com/cyauridev/taskflow_back
Abrir el proyecto en Visual Studio Code
(Este proyecto está desarrollado con NestJS)

EN EL TERMINAL DE VS CODE
=========================
Instalar las dependencias:
npm install

Ejecutar el backend:
npm run start:dev

Base de datos
==============
La base de datos está alojada en Neon (PostgreSQL en la nube), por lo que:

No necesitas crearla
No necesitas instalar PostgreSQL localmente
